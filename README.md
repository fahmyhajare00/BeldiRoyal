# 🍽️ BeldiRoyal

BeldiRoyal est un site web statique conçu pour représenter un restaurant marocain. Il met en valeur les spécialités culinaires marocaines à travers une interface moderne, élégante et responsive.

## ✨ Fonctionnalités

* 🏠 Page d'accueil attractive
* 🍲 Présentation des spécialités marocaines
* 📖 Menu du restaurant
* 🖼️ Galerie de plats
* ℹ️ Section À propos
* 📞 Page de contact
* 📱 Design responsive pour tous les appareils

## 🛠️ Technologies utilisées

* HTML5
* CSS3
* JavaScript

## 🚀 Installation

1. Cloner le dépôt :

   ```bash
   git clone https://github.com/fahmyhajare00/BeldiRoyal.git
   ```

2. Ouvrir le dossier du projet.

3. Lancer le fichier `index.html` dans votre navigateur.

## 🎯 Objectif

Ce projet a été développé pour créer une vitrine web moderne pour un restaurant marocain, mettant en avant son identité, son menu et ses spécialités tout en offrant une navigation simple et une expérience utilisateur agréable.
